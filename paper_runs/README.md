# Paper Runs

Author: Adithya Srinivasan

`paper_runs/` stores generated configs for three-machine evaluation runs.

## Contents

- `configs/`: YAML files generated by `scripts/generate_paper_configs.py`.

Each config is meant to be used by all three machines for one run. Start nodes 1
and 2 with `--node-only`, then start node 0 as the coordinator.
